# Tahoe Kings Dealership
A full-stack web application for managing car inventory.

## Tech Stack
- **Frontend:** React (Vite)
- **Backend:** PHP (PDO)
- **Database:** MySQL
- **Environment:** XAMPP / macOS / Linux

## Setup
1. Clone the repo.
2. Create the database using the SQL file in `/database`.
3. Configure `backend/config/db.php`.
